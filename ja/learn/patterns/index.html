インナーソース パターン

インナーソースのベストプラクティスは、理解しやすく、再利用しやすい形式で体系化されています。

InnerSource Commons は、インナーソース パターンという形で、コミュニティがお互いの経験を共有し、学ぶための場を提供しています。

インナーソースパターン - InnerSource のベスト プラクティスと呼ばれる特定の形式でまとめられたもので、コンテキストでの理解・評価・適用を容易にします。

ワーキンググループの活動


コミュニティコールやSlackでのディスカッションなど、様々な場面でパターンの特定を促進します。

パターンをレビューし、そのパターンが使用された例を提供することで、コミュニティの成熟を支援しています。

GitHub で企業がパターンを共同開発する場を提供しています

インナーソースパターンブックでは最も成熟したパターンを公開しています。

インナーソースパターンマインドマップなどの探索ツールを提供しています。

How to Get Involved

もしあなたが既に社内でインナーソースを使っていて、自分の経験を共有したり、新しいパターンを紹介したりして、上記の取り組みに貢献したいとお考えなのであれば、ぜひ本への投稿や、#innersource-patterns チャンネルでのSlack会話への参加をお待ちしています。
